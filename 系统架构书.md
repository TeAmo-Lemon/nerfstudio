# 🌟 系统架构设计书：SemGeo-FGW Splatting
**(面向 FGW 的结构感知 3D–2D 拓扑对齐 3DGS 风格化系统)**

## 核心思想 (Core Idea)
本系统彻底摒弃了传统 3D 风格化中依赖 2D 分割掩码（如 SAM）和简单特征余弦距离匹配的范式。其核心创新在于：将 3D 场景的局部区域与 2D 风格图的局部纹理，均抽象为**带有语义特征的度量测度空间 (Metric Measure Space)**。通过引入并加速**融合格罗莫夫-瓦瑟斯坦距离 (Fused Gromov-Wasserstein, FGW)**，实现跨模态的纯结构拓扑对齐（例如：基于放射状拓扑，将树冠自动匹配给漩涡纹理）。

---

## 🔷 阶段一：3D 混合特征蒸馏与结构感知分解
**目标：** 在无 2D 掩码条件下，将 3DGS 场景无监督地分解为一组拓扑一致的 3D 度量测度空间 $\mathcal{O} = \{O_1, ..., O_N\}$。

1.  **DINOv2 特征 3D 蒸馏：**
    * 提取训练视角的 DINOv2 特征（1024维），使用 PCA 降维至 16 维（$F_{2D}^{low}$）。
    * 为每个高斯点引入属性向量 $f_{dino} \in \mathbb{R}^{16}$。
    * 在基础训练中加入特征渲染损失：$L_{feat} = ||\text{Rendered}(f_{dino}) - F_{2D}^{low}||^2$。
2.  **构建混合特征空间 $V_{enhanced}$：**
    * 结合几何特征 $g_i = [x,y,z,n_x,n_y,n_z,SH_0]$ 与语义特征 $f_{dino,i}$。
    * 进行标准化与白化（Whitening）处理，平衡几何与语义的度量权重：$V_i = \text{Whiten}([w_g \cdot g_i, w_d \cdot f_{dino,i}])$。
3.  **构建自适应邻域图与图区域生长 (Adaptive Neighborhood + Region Growing)：**
        * **有效点筛选与采样：** 先按不透明度阈值筛选高斯点（默认 $\sigma(\alpha) \ge 0.05$），若点数过大则按不透明度优先保留前 `max_sample_points`，降低后续构图成本。
        * **局部尺度估计（非固定 KNN）：** 对每个点 $i$ 计算局部尺度
            $$
            s_i = d_i^{(k_{base})},\quad k_{base}=8
            $$
            其中 $d_i^{(k_{base})}$ 为第 $k_{base}$ 近邻距离。
        * **自适应半径定义：**
            $$
            r_i = \alpha \cdot s_i,\quad \alpha=2.0
            $$
            并基于“半径约束 + 邻居上限”构建混合邻域。
        * **混合邻域构建（半径内 + capped KNN）：** 先取最多 $k_{max}$ 个候选近邻（$k_{max}=\max(\texttt{knn\_k},32)$），再保留满足
            $$
            d(i,j) < r_i
            $$
            的邻居。
        * **语义门控（Feature Gating）：** 在邻域构建阶段进一步要求 DINO 特征余弦相似度
            $$
            \cos(f_i^{dino},f_j^{dino}) > 0.6
            $$
            以提前切断跨语义连接。
        * **图对称化：** 若 $i\to j$ 或 $j\to i$ 任一成立，则保留无向边 $(i,j)$，提升图连通稳定性。
        * **法线估计与混合特征：** 使用局部近邻估计法线后，将几何项与语义项标准化并加权拼接：
            $$
            h_i = \text{Norm}([w_g\cdot \hat g_i,\; w_d\cdot \hat f_i^{dino}])
            $$
            用于后续生长判定。
        * **区域生长（BFS）：** 从高不透明度点作为种子扩张，仅接受同时满足以下条件的邻居：
            1) 特征相似度阈值（`feature_similarity_threshold`）
            2) 法线一致性阈值（`normal_similarity_threshold`）
            3) 空间约束 $d(i,j) \le r_i\cdot \texttt{radius\_scale}$
        * **标签传播与去噪：** 对小簇（小于 `min_cluster_size`）执行标签传播修复，减少孤立碎片，得到连续语义簇 $C_{3D,j}$。
        * **复杂度优化：** 邻域检索采用分块距离计算（chunked cdist），避免一次性全量距离矩阵带来的显存/内存峰值。
4.  **构建 3D 目标度量空间 $O_n$：**
    * 对每个簇 $C_{3D,j}$ 使用最远点采样 (FPS) 提取代表点集 $X$。
    * 构建图测地距离矩阵 $d_X$（结合 Nyström 扩展进行加速）。
    * 构建概率测度 $\mu_X$ 与代表特征 $F^X$。
    * **输出：** $O_n = (X, d_X, \mu_X, F^X)$。

## 🔷 阶段二：2D 风格图纹理基元分解
**目标：** 将单张风格图无监督地分解为一组局部纹理基元的度量测度空间 $\mathcal{M} = \{M_1, ..., M_K\}$。

1.  **Patch 级特征提取：** 对风格图提取 DINOv2 特征，将每个 Patch 视为点集 $Y$ 中的一个点。
2.  **空间拓扑图构建：** 基于 4/8 邻域和特征相似度构建 2D 像素图。
3.  **纹理基元聚类：** 对图节点进行无监督聚类（此处可使用 KMeans 或谱聚类），划分出不同的纹理区块（如：天空条纹区、地面斑块区）。
4.  **构建 2D 目标度量空间 $M_k$：**
    * 定义空间距离矩阵 $d_Y$。
    * **输出：** $M_k = (Y, d_Y, \mu_Y, F^Y)$。

## 🔷 阶段三：FGW 跨域拓扑对齐与计算加速 (核心创新)
**目标：** 为每个 3D 区域 $O_n$ 自动匹配最符合其结构逻辑的 2D 纹理 $M_k$。

1.  **定义联合匹配代价：**
    * **特征代价 (Feature Cost)：** 衡量基础语义/颜色的余弦距离 $L_{feat} = 1 - \cos(F^X, F^Y)$。
    * **结构代价 (Structure Cost)：** 衡量内部拓扑的异构差异 $L_{struct} = |d_X - d_Y|^2$。
    * 总目标：$\min_{\pi} \sum L_{feat} \pi + \gamma \sum L_{struct} \pi \otimes \pi$。
2.  **大尺度计算加速 (Sinkhorn-FGW + Coarse-to-Fine)：**
    * **熵正则化：** 引入 KL 散度 $-\epsilon H(\pi)$，利用 Sinkhorn 迭代在 GPU 上将 $O(N^3)$ 的匹配复杂度降至 $O(N^2)$。
    * **由粗到细：** 先利用极少数 Super-nodes 确定宏观匹配关系，再在局部子区域进行细粒度 FGW 计算，彻底解决显存爆炸问题。
3.  **目标赋值：** 经过匈牙利匹配 (Hungarian Matching)，将对应的 2D 基元特征赋给对应的 3D 高斯点簇，作为优化目标 $F_i$。

## 🔷 阶段四：多视角一致性风格生成 (NNFM)
**目标：** 冻结场景几何，通过 3D 优化实现无闪烁的风格迁移。

1.  **隔离优化：** 仅更新球谐函数 (SH) 系数和基础颜色，完全冻结高斯点坐标 $xyz$ 和协方差矩阵参数。
2.  **最近邻特征匹配 (NNFM Loss)：**
    * 多视角渲染当前场景，提取 VGG 特征 $\Phi(I_{render})$。
    * 最小化渲染特征与匹配到的目标纹理特征的格拉姆矩阵 (Gram Matrix) 差异。
    * **关键约束 (Stop-Gradient)：** 目标特征 $\Phi(M_k)$ 和高斯点内部的 $f_{dino}$ 必须 detach，防止梯度回传破坏几何场。
3.  **法线感知一致性 (Normal-Aware Consistency)：**
    * 在损失函数中增加惩罚项：强制法线夹角极小（共面）的相邻高斯点保持相近的风格特征表达，从物理层面杜绝视点移动时的“纹理滑动 (Flickering)”。
