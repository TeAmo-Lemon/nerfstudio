# 🌟 系统架构设计书：SemGeo-FGW Splatting
**(面向 FGW 的结构感知 3D–2D 拓扑对齐 3DGS 风格化系统)**

## 核心思想 (Core Idea)
本系统彻底摒弃了传统 3D 风格化中依赖 2D 分割掩码（如 SAM）和简单特征余弦距离匹配的范式。其核心创新在于：将 3D 场景的局部区域与 2D 风格图的局部纹理，均抽象为**带有语义特征的度量测度空间 (Metric Measure Space)**。通过引入并加速**融合格罗莫夫-瓦瑟斯坦距离 (Fused Gromov-Wasserstein, FGW)**，实现跨模态的纯结构拓扑对齐（例如：基于放射状拓扑，将树冠自动匹配给漩涡纹理）。

---

## 🔷 阶段一：3D 混合特征蒸馏与结构感知分解
**目标：** 在无 2D 掩码条件下，将 3DGS 场景无监督地分解为一组拓扑一致的 3D 度量测度空间 $\mathcal{O} = \{O_1, ..., O_N\}$。

### 1.1 DINOv2 2D 特征预处理（离线阶段）

在开始 3D 训练之前，先对训练数据集进行特征预提取和降维。

1. **特征提取：** 使用预训练 DINOv2（推荐 `dinov2_vitl14`）对所有训练视角图像提取 Feature Map，得到每张图的特征图 $F_{2D} \in \mathbb{R}^{H \times W \times 1024}$。

2. **全局 PCA 降维（关键）：**
   - DINOv2 原始特征维度（1024）过高，直接存入高斯点会导致显存爆炸且渲染极慢。
   - **做法：** 收集所有训练图像的特征向量（采样或全量），运行一次全局 PCA，将维度从 1024 压缩至 **16 维**。
   - **依据：** 8–16 维足以保留 DINOv2 的核心语义信息（主成分累积方差 > 80%），同时极大加速后续聚类和渲染。
   - **保存：** 将降维后的特征图 $F_{2D}^{low} \in \mathbb{R}^{H \times W \times 16}$ 保存为 Tensor，作为后续 3D 蒸馏的真值 (Ground Truth)。

### 1.2 3DGS 模型扩展

修改 3DGS 底层数据结构，让每个高斯点同时携带颜色和语义信息。

1. **参数扩展：** 在 GaussianModel 中为每个高斯点增加属性向量 $f_{dino} \in \mathbb{R}^{16}$，形状为 `[N, 16]`。

2. **特征渲染：**
   - **原理：** 与颜色渲染一致，使用 Alpha Blending 对特征进行加权合成：
     $$F_{feat}^{render} = \sum_{i \in N} f_{dino,i} \cdot \alpha_i \prod_{j=1}^{i-1} (1 - \alpha_j)$$
   - **实现方案（按优先级）：**
     1. *(推荐)* 复用现有 rasterizer 的多通道渲染能力（gsplat / nerfstudio 已支持渲染任意 attribute），将 `features_dino` 作为额外通道传入；
     2. *(备选)* 若需极致性能，可修改 CUDA forward/backward kernel，增加特征渲染通道。

3. **法线估计：** 从每个高斯点的协方差矩阵中提取最小特征向量作为法线 $N_i = [n_x, n_y, n_z]$，用于后续几何特征构建。

### 1.3 特征蒸馏训练

在基础 3DGS 重建训练的同时进行特征蒸馏。

- **联合损失函数：**
  $$Loss_{total} = Loss_{rgb} + \lambda_{feat} \cdot Loss_{feat}$$
  $$Loss_{feat} = \| F_{feat}^{render} - F_{2D}^{low} \|^2$$

- **超参设置：** $\lambda_{feat}$ 建议设为 **0.01 ~ 0.1**，防止特征梯度干扰几何重建。可随训练进度线性衰减。

- **结果：** 训练完成后，每个高斯点携带一个代表其语义类别的 16 维编码 $f_{dino,i}$。

### 1.4 构建增强混合特征空间 $V_{enhanced}$

将几何特征与语义特征融合，构建用于聚类的增强特征向量。

1. **几何分量 $g_i$：**
   $$g_i = [x, y, z,\; n_x, n_y, n_z,\; SH_0]$$
   其中 $[x,y,z]$ 为空间坐标，$[n_x,n_y,n_z]$ 为法线方向，$SH_0$ 为基础颜色（球谐直流分量）。

2. **语义分量 $f_{dino,i}$：** 蒸馏后的 16 维 DINO 特征。

3. **标准化与白化：**
   - 对几何分量和语义分量分别进行 Z-score 标准化：$\text{Norm}(x) = (x - \mu) / \sigma$。
   - 对拼接后的向量进行 **白化 (Whitening)** 处理，消除特征维度间的相关性，平衡几何与语义在距离度量中的贡献：
     $$V_i = \text{Whiten}([w_g \cdot \text{Norm}(g_i),\; w_d \cdot \text{Norm}(f_{dino,i})])$$

4. **加权策略：**
   - 若希望分割更贴合物理边界（如区分相邻小物体），增大几何权重 $w_g$（推荐 $w_g = 1.0$）。
   - 若希望跨空间聚合同类语义物体，增大语义权重 $w_d$（推荐 $w_d = 0.5 \sim 1.0$）。
   - 默认均衡配置：$w_g = 1.0,\; w_d = 0.8$。

### 1.5 粗粒度预聚类（FAISS GPU 加速，可选）

当高斯点数达到百万级时，直接构建全量邻域图代价过高。先用 FAISS 进行粗聚类可大幅降低后续图计算的规模。

1. **FAISS GPU K-Means：**
   - 输入所有有效高斯点的 $V_{enhanced}$。
   - 设定聚类数 $K = 50 \sim 100$（有意过分割，后续可通过图生长合并）。
   - FAISS 是目前最快的 GPU 向量搜索库，百万级点云聚类可在秒级完成。

2. **输出：** 每个高斯点的粗聚类标签 $L_i^{coarse}$，以及每个粗簇的点集。后续的图构建和区域生长可在各粗簇内独立并行执行，将复杂度从 $O(N^2)$ 降至 $O(\sum_k N_k^2) \ll O(N^2)$。

> **注：** 若场景点数较少（< 10万），可跳过此步骤，直接进入 1.6 的全量图构建。

### 1.6 自适应邻域图构建

在粗簇内部（或全量点集上），构建保留局部拓扑结构的邻域图。

1. **有效点筛选与采样：**
   - 按不透明度阈值筛选：$\sigma(\alpha) \ge 0.05$（过滤低贡献高斯点）。
   - 若点数仍过大，按不透明度降序保留前 `max_sample_points` 个点（建议 `max_sample_points = 200,000`）。

2. **局部尺度估计（非固定 KNN）：**
   - 对每个点 $i$，计算其到第 $k_{base}$ 近邻的距离作为局部尺度：
     $$s_i = d_i^{(k_{base})},\quad k_{base} = 8$$

3. **自适应半径邻域：**
   - 定义自适应半径 $r_i = \alpha \cdot s_i$，$\alpha = 2.0$。
   - 先取最多 $k_{max} = \max(knn\_k, 32)$ 个候选近邻。
   - 仅保留满足 $d(i,j) < r_i$ 的邻居。

4. **语义门控 (Feature Gating)：**
   - 在邻域构建阶段提前切断跨语义连接：
     $$\cos(f_i^{dino}, f_j^{dino}) > 0.6$$
   - 这能有效防止语义差异大的区域被错误连接。

5. **图对称化：** 若 $i \to j$ 或 $j \to i$ 任一方向的边存在，则保留无向边 $(i,j)$，提升图的连通稳定性。

6. **复杂度优化：** 邻域检索采用分块距离计算 (chunked cdist)，避免一次性全量距离矩阵带来的显存/内存峰值。块大小建议 4096。

### 1.7 图区域生长 (BFS Graph Region Growing)

从种子点出发，通过广度优先搜索 (BFS) 将图划分为语义一致的连通区域。

1. **种子选择：** 按不透明度降序选择未访问点作为种子。

2. **生长接受条件（三个条件同时满足）：**
   - **特征相似度：** $\cos(h_i, h_j) > \tau_{feat}$（$\tau_{feat}$ 建议 0.7~0.85）
   - **法线一致性：** $|n_i \cdot n_j| > \tau_{normal}$（$\tau_{normal}$ 建议 0.6~0.8）
   - **空间约束：** $d(i,j) \le r_i \cdot \gamma$（$\gamma$ 建议 1.0~1.5）

   其中 $h_i$ 为 1.4 节中构建的混合特征向量。

3. **终止条件：** 当队列为空（无可接受的邻居）时，当前区域生长结束，选择下一个未访问种子点继续。

### 1.8 后处理：去噪与合并

初始分割可能存在碎片和噪声，需要后处理优化。

1. **标签传播 (Label Propagation)：**
   - 对小簇（点数 < `min_cluster_size`，建议 50~200）执行标签传播：将小簇中每个点的标签替换为其 K=10 近邻中出现最多的标签。
   - 这能有效修复孤立的误分点，减少碎片。

2. **KNN 投票平滑 (可选)：**
   - 对每个点检查其最近 K=10 个邻居。若超过 80% 的邻居为 Label A 而自身为 Label B，则更正为 A。
   - 能消除点云内部的椒盐噪声。

3. **连通分量分析 (Connected Components)：**
   - 对同一标签但空间上不连通的区域，拆分为不同的实例 ID。
   - 通过空间距离阈值（如 $2 \times$ 平均邻域距离）判断连通性。
   - 这解决了"两个相同语义但物理分离的物体被标为同一标签"的问题。

### 1.9 构建 3D 目标度量空间 $O_n$

对每个最终语义簇 $C_{3D,j}$，构建用于 FGW 匹配的度量测度空间。

1. **代表点采样：** 使用最远点采样 (FPS) 从簇中提取 $m$ 个代表点 $X = \{x_1, ..., x_m\}$（建议 $m = 50 \sim 200$）。

2. **图测地距离矩阵 $d_X$：**
   - 在代表点之间构建 KNN 图（$k = 5 \sim 10$）。
   - 计算最短路径距离作为测地距离近似。
   - *(可选加速)* 使用 Nyström 扩展从采样点子矩阵近似全量测地距离。

3. **概率测度 $\mu_X$：** 基于各代表点邻域内的不透明度之和归一化得到：
   $$\mu_X(i) = \frac{\sum_{p \in \mathcal{N}(x_i)} \alpha_p}{\sum_{j=1}^m \sum_{p \in \mathcal{N}(x_j)} \alpha_p}$$

4. **代表特征 $F^X$：** 对各代表点邻域内的 DINO 特征取加权平均。

5. **输出：** $O_n = (X, d_X, \mu_X, F^X)$，即一个带语义特征的 3D 度量测度空间。

## 🔷 阶段二：2D 风格图纹理基元分解
**目标：** 将单张风格图无监督地分解为一组局部纹理基元的度量测度空间 $\mathcal{M} = \{M_1, ..., M_K\}$。

1.  **Patch 级特征提取：** 对风格图提取 DINOv2 特征，将每个 Patch 视为点集 $Y$ 中的一个点。
2.  **空间拓扑图构建：** 基于 4/8 邻域和特征相似度构建 2D 像素图。
3.  **纹理基元聚类：** 对图节点进行无监督聚类（可使用 KMeans 或谱聚类），划分出不同的纹理区块（如：天空条纹区、地面斑块区）。
4.  **构建 2D 目标度量空间 $M_k$：**
    * 定义空间距离矩阵 $d_Y$。
    * **输出：** $M_k = (Y, d_Y, \mu_Y, F^Y)$。

## 🔷 阶段三：FGW 跨域拓扑对齐与计算加速 (核心创新)
**目标：** 为每个 3D 区域 $O_n$ 自动匹配最符合其结构逻辑的 2D 纹理 $M_k$。

1.  **定义联合匹配代价：**
    * **特征代价 (Feature Cost)：** 衡量基础语义/颜色的余弦距离 $L_{feat} = 1 - \cos(F^X, F^Y)$。
    * **结构代价 (Structure Cost)：** 衡量内部拓扑的异构差异 $L_{struct} = |d_X - d_Y|^2$。
    * 总目标：$\min_{\pi} \sum L_{feat} \pi + \gamma \sum L_{struct} \pi \otimes \pi$。
2.  **大尺度计算加速 (Sinkhorn-FGW + Coarse-to-Fine)：**
    * **熵正则化：** 引入 KL 散度 $-\epsilon H(\pi)$，利用 Sinkhorn 迭代在 GPU 上将 $O(N^3)$ 的匹配复杂度降至 $O(N^2)$。
    * **由粗到细：** 先利用极少数 Super-nodes 确定宏观匹配关系，再在局部子区域进行细粒度 FGW 计算，彻底解决显存爆炸问题。
3.  **目标赋值：** 经过匈牙利匹配 (Hungarian Matching)，将对应的 2D 基元特征赋给对应的 3D 高斯点簇，作为优化目标 $F_i$。

## 🔷 阶段四：多视角一致性风格生成 (NNFM)
**目标：** 冻结场景几何，通过 3D 优化实现无闪烁的风格迁移。

1.  **隔离优化：** 仅更新球谐函数 (SH) 系数和基础颜色，完全冻结高斯点坐标 $xyz$ 和协方差矩阵参数。
2.  **最近邻特征匹配 (NNFM Loss)：**
    * 多视角渲染当前场景，提取 VGG 特征 $\Phi(I_{render})$。
    * 最小化渲染特征与匹配到的目标纹理特征的格拉姆矩阵 (Gram Matrix) 差异。
    * **关键约束 (Stop-Gradient)：** 目标特征 $\Phi(M_k)$ 和高斯点内部的 $f_{dino}$ 必须 detach，防止梯度回传破坏几何场。
3.  **法线感知一致性 (Normal-Aware Consistency)：**
    * 在损失函数中增加惩罚项：强制法线夹角极小（共面）的相邻高斯点保持相近的风格特征表达，从物理层面杜绝视点移动时的"纹理滑动 (Flickering)"。

## 实现路线图建议

1. **第一阶段（验证可行性）：** 在小规模场景（< 5万高斯点）上实现 1.1→1.4→1.6→1.7→1.9，验证从特征蒸馏到度量空间构建的完整链路。
2. **第二阶段（扩展到大规模）：** 加入 1.5 FAISS 粗聚类和分块距离计算，处理百万级高斯点场景。
3. **第三阶段（FGW 匹配）：** 实现阶段二和阶段三，验证跨域拓扑对齐效果。
4. **第四阶段（风格生成）：** 实现阶段四的风格优化管线，完成端到端系统。
